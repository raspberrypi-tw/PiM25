# airbox-pi
## create a airbox to detect pm2.5 with raspberry pi

#                    GPIO  3  |  GND      G5T GND            #
#                    GPIO  4  | UART TX                      #
#                     GND     | UART RX                      #
#                    GPIO 17  | GPIO 18   Wifi_Reset         #
#                    GPIO 27  |  GND      GPS NEO6 GND       #
#                    GPIO 22  | GPIO 23   G5T Tx             #
#                     +3.3V   | GPIO 24   GPS NEO6 Tx        #
#                    GPIO 10  |  GND                         #
#                    GPIO  9  | GPIO 25                      #
#                    GPIO 11  | GPIO  8                      #
#                     GND     | GPIO  7                      #
#                    Reserved | Reserved                     #
#                    GPIO  5  |  GND                         #
#                    GPIO  6  | GPIO 12                      #
